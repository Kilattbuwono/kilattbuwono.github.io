Beranda Admin Penjualan Barang!

Aplikasi ini dirancang untuk membantu Anda mengelola data penjualan barang dengan mudah. Berikut adalah langkah-langkah dasar untuk menggunakan aplikasi ini:

  1. Navigasikan ke halaman **Transaksi** untuk mencatat penjualan baru.
  2. Isi formulir transaksi dengan detail yang lengkap (Nama Transaksi, Jumlah, Kategori).
  3. Klik tombol **Simpan** untuk menyimpan transaksi Anda.
  4. Lihat daftar transaksi yang sudah dicatat di halaman **Daftar Transaksi** (link dari halaman Transaksi).
  5. Kunjungi halaman **Profil** untuk mengetahui deskripsi lebih lanjut tentang aplikasi ini.
  6. Untuk akses aman, pastikan Anda sudah **Login**.
